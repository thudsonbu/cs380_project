Welcome to Part I of Assignment 2, where you are currently on the index page for the project.
Here, I developed a webpage that is Bentley University themed, utilizing the brand colors of blue, black and white.

For Part II, I created a table in MySQL Workbench defined as Persons. This table contains 5 elements, a LoginID which is the primary key, first and last name, a picture URL and a bio. After creating this table, I developed a Persons.php file that allows the connection of the SQL table to the HTML file. With that, we utilize a CSS linked stylesheet to make the table more creative and styled.

For Part III, I utilized PHP to import two text files called "boynames.txt" and "girlnames.txt". With this, I compared and analyzed the names within the files to identify any similarities.

For the final part, I develop another table in MySQL Workbench defined as Product, where it contains 5 elements, a ProductID which is the primary key, Product Name, Description, picture URL and the price of the item.
